# C-NAPSE 2026 매칭 카드

계명 CMF 2026 시즌 시냅스 매칭 카드 & 포스터 뷰어.

## 로컬 실행
저장소 루트에서:
```bash
python3 -m http.server 8000
```
브라우저에서 http://localhost:8000 접속.

## 배포 (GitHub Pages)
1. 이 저장소를 GitHub에 푸시
2. Settings → Pages → Source: **main branch / (root)** 선택
3. 몇 분 후 `https://<user>.github.io/<repo>/` 에서 접속 가능
